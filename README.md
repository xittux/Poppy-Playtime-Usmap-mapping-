Poppy Playtime – Chapter 5 USMAP Mapping

This repository contains the USMAP mapping file(s) for Chapter 5 of Poppy Playtime.

These mappings are provided to help correctly interpret Unreal Engine serialized data such as classes, structs, and property definitions related to this chapter.

Purpose

This repository is shared for educational and curiosity purposes, especially for 3D artists and animators working with tools such as Blender or Autodesk Maya who are interested in understanding how Unreal Engine structures its data.

It does not include any original game assets and is not affiliated with or endorsed by the developers of Poppy Playtime. The goal is purely technical exploration and learning.
